# BolittosApi
Projeto ASP.NET Core (Web API) com EF Core e SQLite para gestao de pedidos

Como usar:
- Requisitos: .NET 8 SDK
- Restaurar pacotes: `dotnet restore`
- Rodar: `dotnet run`
- Swagger: `https://localhost:{port}/swagger`

O projeto já contém `EnsureCreated()` para criar o banco SQLite (`bolitos.db`) automaticamente.
